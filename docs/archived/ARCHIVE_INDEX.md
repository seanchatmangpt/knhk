## Archive Policy

Files are archived when:
1. **Consolidated** - Content has been merged into 80/20 consolidated guides
2. **Historical** - Status reports, certifications, or validation reports that are no longer current
3. **Detailed Reference** - Detailed reference docs that are less critical (80/20 principle)

## Accessing Archived Documentation

- **Primary Entry Point**: Use the consolidated 80/20 guides in the main `docs/` directory
- **Detailed Reference**: Check `reference-docs/` for comprehensive reference documentation
- **Historical Context**: Check `historical-reports/` for historical status and validation reports

## Note

All archived documentation is kept for reference but is not actively maintained. For current information, always refer to the consolidated 80/20 guides.
