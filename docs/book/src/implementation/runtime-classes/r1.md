# R1 Hot Path

R1 (Hot Path) runtime class for ≤8 tick operations.

## Overview

R1 provides:
- ≤8 ticks per operation
- Branchless operations
- SIMD-optimized
- Zero-copy operations

## Performance Targets

- **Latency**: ≤2ns (8 ticks)
- **Operations**: Branchless
- **Memory**: Cache-aligned (64-byte)
