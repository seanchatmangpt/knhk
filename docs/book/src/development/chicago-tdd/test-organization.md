# Test Organization

Test organization for Chicago TDD.

## Overview

Tests are organized by subsystem:
- `chicago_tdd_beat_scheduler.rs` - Beat scheduler tests
- `chicago_tdd_pipeline.rs` - ETL pipeline tests
- `chicago_tdd_ring_conversion.rs` - Ring conversion tests
- `chicago_tdd_hook_registry.rs` - Hook registry tests
- `chicago_tdd_runtime_class.rs` - Runtime class tests
