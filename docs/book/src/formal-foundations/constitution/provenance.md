# Provenance: hash(A) = hash(μ(O))

The provenance law: **hash(A) = hash(μ(O))** - Every assertion is cryptographically linked to its operation.

## Overview

Every assertion (A) is cryptographically linked to its operation (O):
- Receipt generation
- Merkle linking
- Cryptographic verification
- Provenance tracking
