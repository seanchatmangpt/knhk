# Epoch Containment: μ ⊂ τ

The epoch containment law: **μ ⊂ τ** (τ=8) - All operations complete within 8 ticks.

## Overview

All hook evaluations must complete within 8 ticks (Chatman Constant):
- Time-bounded execution
- Guard constraints
- Performance budgets
- SLO monitoring
