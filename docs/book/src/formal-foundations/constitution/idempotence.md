# Idempotence: μ ∘ μ = μ

The idempotence law: **μ ∘ μ = μ** - Hook evaluation is idempotent.

## Overview

Hook evaluation is idempotent:
- Applying μ twice is equivalent to applying it once
- Enables safe retries
- Enables parallel execution
