# Receipt Verification Example

This example demonstrates receipt verification and merging.

## Files

- `verify.sh` - Receipt verification script
- `README.md` - This file

## Usage

```bash
./verify.sh
```

## Steps

1. Get receipt by ID
2. Verify receipt integrity
3. Merge multiple receipts
4. Verify merged receipt

## Expected Output

- Receipt details
- Verification results
- Merged receipt information
