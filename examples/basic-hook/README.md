# Basic Hook Example

This example demonstrates basic hook execution with both hot path and warm path operations.

## Files

- `hook.ttl` - Hook definition file
- `run.sh` - Execution script
- `README.md` - This file

## Usage

```bash
./run.sh
```

## Example Operations

1. **Hot Path Hook** (ASK_SP) - ≤8 ticks
2. **Warm Path Hook** (CONSTRUCT8) - ≤500ms

## Expected Output

- Hook execution results
- Receipt information
- Path routing (hot vs warm)
