# knhk-otel Documentation

OpenTelemetry integration for observability.

## Overview

The `knhk-otel` crate provides:
- Span ID generation
- Metrics collection
- Tracing support
- Receipt provenance tracking

## Architecture

- **Span Generation**: OTEL-compatible span IDs
- **Metrics**: Query latency, cache hits, etc.
- **Tracing**: Distributed tracing support
