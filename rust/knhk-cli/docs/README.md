# knhk-cli Documentation

Command-line interface for KNHK operations.

## Overview

The `knhk-cli` crate provides CLI commands for:
- System initialization (`boot init`)
- Connector management (`connect register`)
- Coverage definition (`cover define`)
- Reflex declaration (`reflex declare`)
- Epoch creation (`epoch create`)
- Pipeline execution (`pipeline run`)
